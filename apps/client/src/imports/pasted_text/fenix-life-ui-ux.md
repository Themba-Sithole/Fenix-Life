# Fenix Life - Complete UI/UX Prototype

Design a modern, premium, highly polished UI/UX prototype for a 2D life and business simulation game called **Fenix Life**.

The game combines life simulation, entrepreneurship, career progression, investing, real estate, education, family management, and wealth building into one immersive experience.

The visual style should feel like a combination of BitLife, Game Dev Tycoon, Startup Company, The Sims UI, and modern fintech applications.

The interface should be clean, elegant, minimalistic, and information-rich without feeling cluttered.

Use a modern design system with consistent spacing, rounded cards, soft shadows, subtle gradients, and professional typography.

Primary Colors:

* Dark Navy (#0B132B)
* Deep Blue (#1C2541)
* Emerald Green (#2EC4B6)
* Gold (#F4B400)
* White (#FFFFFF)
* Light Gray (#F5F7FA)

The visual language should communicate wealth, growth, professionalism, innovation, and long-term success.

## Design Philosophy

Every screen should feel premium, modern, realistic, and easy to navigate. The interface should resemble software used by CEOs, investors, and financial professionals while remaining approachable for casual players. Prioritize clarity, hierarchy, responsiveness, and consistency. Use subtle animations, polished cards, elegant data visualizations, and intuitive layouts that make players feel like they are managing both a successful life and a global business empire.
